# GAMEPLAY

# What is GamePlay?

GamePlay is a frontend for MS-DOS, Windows 3.1, Windows 95/98, ScummVM, Apple II, Atari 800, Atari ST, Amiga, and DSP Emulator games. It allows you to play games easily and quickly, without complications and without needing to configure anything. Simply choose a game from the list and start playing — everything is preconfigured (best audio, best graphics, extra parameters, etc.).

![Main Screen](https://i.ibb.co/V0jXJtKb/gameplay-082-en.png)

## What games can I play?

Initially, there is a list of:
- 378 MS-DOS games
- 5 Windows 3.1 games
- 4 Windows 95/98 games
- 49 ScummVM games (2 exclusives)
- 119 Apple II games
- 41 Atari 800 games
- 38 Atari ST games
- 123 Amiga games

All of them are ready to play and include preview images, manuals, maps, and guides so you can fully enjoy the games.

There are also more than 400 Arcade, Console, and 8-bit computer games ready to play.

You do not need to configure anything — no audio, no video, nothing. Just enjoy.

If the games are not available, GamePlay allows you to download them automatically.

## Can I add more games?

Of course! If you have favorite MS-DOS games that are not included in the list, you can easily and quickly add your own games.

In the advanced options, you can:
- Use ZIP files
- Add CD-ROM images
- Change memory settings
- Select video cards
- Add manuals, guides, maps, and more

## What languages does GamePlay support?

The language is detected automatically and supports:
- Spanish
- English
- German
- French
- Italian

If none of these languages are detected, English will be selected by default.
